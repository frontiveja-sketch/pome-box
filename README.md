# POME BOX

作ったゲームと、くだらないブログの箱。

## 構成

- `index.html` — サイト本体（HTML/CSS/JSすべて1ファイル、ビルド不要）

## 公開方法（GitHub Pages）

1. このリポジトリの **Settings → Pages** を開く
2. **Source: Deploy from a branch / Branch: main / (root)** を選んで Save
3. 数分後に `https://frontiveja-sketch.github.io/pome-box/` で公開されます

## 更新のしかた

- ゲームを追加: `index.html` の GAMES セクションに `game-card` をコピーして編集
- ブログを追加: BLOG セクションに `b-entry` をコピーして編集

## クレジット

- フォント: Google Fonts（DotGothic16 / Zen Kaku Gothic New / Archivo）
